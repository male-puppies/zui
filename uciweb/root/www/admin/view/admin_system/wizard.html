设置向导

如果您对网络知识或这个产品不太熟悉，您可以通过这个向导提示轻松完成上网所需的基本网络参数。如果您是专家，您也可以退出这个向导程序，进入左侧菜单项中选择您需要修改的设置。

要继续设置向导，请单击“下一步”。
要退出设置向导，请单击“退出向导”。

选择上网方式

三种不同的上网方式，请您根据自身情况进行选择。

PPPoE拨号

您申请宽带服务时候，网络服务商提供给您的上网帐号和密码，请对应填入下框。如您遗忘或不太清楚，请咨询您的网络服务商。

静态IP获取

您申请宽带服务，并具有固定IP地址时，网络服务商提供给您的一些基本网络参数，请对应填入下框。如您遗忘或不太清楚，请咨询您的网络服务商。

无线配置

配置无线网络的基本配置，详细配置请在左侧无线管理的WLAN管理进行配置。
您当前未接入任何AP，当接入AP选择生效AP列表后，该配置才能生效。
